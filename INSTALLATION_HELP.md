# WindowSnap installation help

Download WindowSnap only from the project's GitHub Releases page. Production ZIP and DMG assets are universal, Developer ID signed, and notarized by Apple.

## Install from DMG

1. Open the downloaded DMG.
2. Drag `WindowSnap.app` to the Applications shortcut.
3. Eject the DMG.
4. Launch WindowSnap from Applications.

## Install from ZIP

1. Expand the downloaded ZIP.
2. Move `WindowSnap.app` to Applications.
3. Launch WindowSnap from Applications.

WindowSnap asks for Accessibility permission because moving and resizing other applications' windows requires it. Follow the in-app explanation and grant access in System Settings.

If macOS rejects an official release, do not disable Gatekeeper or remove security metadata. Verify that the filename and checksum match the GitHub release, then open an issue with the release filename, checksum, Mac architecture, and macOS version.
